# 📊 Financial Data Cleaning & Transformation Project (Microsoft Excel)

## 📌 Project Overview

This project demonstrates an end-to-end data cleaning and transformation workflow using Microsoft Excel. The objective was to convert a messy financial dataset containing **25,000+ records** into a clean, structured, and analysis-ready dataset suitable for reporting, dashboards, SQL, and Power BI.

---

## 🎯 Objectives

* Improve data quality and consistency
* Remove duplicate records
* Standardize formatting across the dataset
* Handle missing and inconsistent values
* Prepare data for business intelligence and analytics

---

## 📂 Dataset Challenges

The raw dataset contained several real-world data quality issues, including:

* Missing client information
* 1,838 duplicate records
* Mixed date formats
* Invalid email addresses
* Inconsistent text formatting
* Missing phone numbers
* Transaction amounts stored as text
* Inconsistent bank names
* Currency inconsistencies
* Blank and incomplete fields

---

## 🛠️ Data Cleaning Process

The following Excel techniques were used:

* Remove Duplicates
* Date Standardization
* TRIM, CLEAN, PROPER, UPPER, and LOWER functions
* Email validation
* Phone number standardization
* Text-to-number conversion
* Missing value handling
* Bank name and currency standardization
* Excel Tables for structured reporting

---

## 📈 Results

* ✅ Cleaned and transformed **25,000+ records**
* ✅ Removed **1,838 duplicate records**
* ✅ Standardized key data fields
* ✅ Improved data quality and consistency
* ✅ Produced an analysis-ready dataset for SQL, Power BI, Excel dashboards, and business reporting
